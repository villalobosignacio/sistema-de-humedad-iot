Diagrama interactivo en Wokwi (incluye ESP32, sensor de humedad y LEDs):

Esquemático en Wokwi:https://wokwi.com/projects/462746890644844545

Componentes conectados
ESP32 NodeMCU
Sensor de humedad resistivo → GPIO34
LED Rojo → GPIO25 + resistencia 220Ω
LED Verde → GPIO26 + resistencia 220Ω  
LED Azul → GPIO27 + resistencia 220Ω
